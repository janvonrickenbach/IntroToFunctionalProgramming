Functional Programming¶

Example: Moving average¶

\begin{equation} avg(i) = \frac{1}{3} \sum_{j=-1}^1 f(i+j)\\ \end{equation}
\begin{align} f(i) &=\begin{cases} g(0) & \text{if $i<0$}\\ g(end) & \text{if $i>end$}\\ g(i) & \text{otherwise} \end{cases} \end{align}

In a numerical context g is a data array that is defined at certain discrete locations i in the interval [0, end]

In [68]:
def avg(data):
    avg_values = []
    for i,value in enumerate(data):
        # Handling left boundary
        if i == 0:
            left_val = value
        else:
            left_val = data[i-1]
            
        # Handling right boundary
        if i == len(data) - 1:
            right_val = value 
        else:
            right_val = data[i+1]
        
        # Actual moving average computation
        avg_value = (right_val + left_val + value)/3.
        
        avg_values.append(avg_value)
    return avg_values

Linear extrapolation at boundaries¶

\begin{align} f(i) & =\begin{cases} g(0) + i \frac{d g}{d i}\vert_{0} & \text{if $i<0$}\\ g(end) + (i-end) \frac{d g}{d i}\vert_{end} & \text{if $i>end$}\\ g(i) & \text{otherwise} \end{cases} \end{align}
In [78]:
def get_fwd_d(g):
    def fwd_dg(i):
        return g(i+1) - g(i)
    return fwd_dg
In [79]:
def get_bwd_d(g):
    def bwd_dg(i):
        return g(i) - g(i-1)
    return bwd_dg
In [80]:
def get_linear_extrapolator(g, end):
    def f(i):
        if i < 0:
            dg_di = get_fwd_d(g)
            return g(0) + i * dg_di(0)  
        elif i > end:
            dg_di = get_bwd_d(g)
            return f(end) + (i-end) * dg_di(end)
        else:
            return g(i)
        
    return f

def get_avg_generic(f, n):
    def avg(i):
        n_vals = (1+2*n)
        return 1./(n_vals) * get_sum(f, i-n, n_vals)
    return avg

def memoize(f):
    memo = {}
    def memoized_f(i):
        if i in memo:
            print("cache hit")
            return memo[i]
        else:
            print("function evaluation")
            memo[i] = f(i)
            return memo[i]
    return memoized_f

Advantages¶

  • Expressive/ Close to mathematical problem definition
  • Testable
  • Modular
  • Lazy evaluation, Declarative (Performance)
  • No mutable state
  • Concurrency

Disadvantages¶

  • Performance (Recursion, Tail Call Optimization, Immutability)
  • Debugging recursive code
  • Language support (also related to performance)
  • Library support
